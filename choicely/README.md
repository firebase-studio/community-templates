## Overview

This is a Firebase Studio template for the Choicely Studio platform â€” a ready-made starting point to create cross-platform native apps connected to Choicely services. 
It's designed for anyone who want to create a new mobile app quickly.

## Requirements

- [A Choicely account](https://studio.choicely.com).
- A Google account.
- No developer tools are required to create a project through Firebase Studio.

## How to use

1. Click the "Try in Firebase Studio" button above.  
2. Give your project a name and provide the required template parameters when prompted.
3. Create the app with the help of Gemini AI chat.

## Template parameters

- `app_key` (required): Your Choicely app key.  
- `api_key` (optional): Optional API key to make the integration easier.

You can obtain these parameters from your Choicely Studio dashboard at https://studio.choicely.com.
